# Project 3 – Full Stack Todo App

First full stack project built during a React course under instructor guidance.

## 🚀 Tech Stack
- React + TypeScript
- Strapi (Backend API)
- React Query
- React Hook Form
- Axios
- JWT Authentication

## ✅ Features
- User Login / Register
- Create / Edit / Delete Todos
- User-specific todos
- Validations on both ends
- Real-time state updates

## 🛠️ Run Locally
```bash
npm install
npm run dev
